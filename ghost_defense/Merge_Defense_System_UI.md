# Merge Defense - System & UI

## 2. 로비 화면 구성

### 2.1 로비 UI 구조 (메인 탭)
게임의 메인 화면은 하단 5개의 탭으로 구성된다.

1. **강화**: 캐릭터 스탯 및 머지 단계 영구 강화
2. **카드**: 보유 카드 확인 및 등장 확률 강화
3. **배틀**: 스테이지 정보 확인 및 게임 시작 (기본 진입 화면)
4. **상점**: 재화 및 패키지 구매
5. **미션**: 일일/업적 미션 및 보상 수령

---

## 17. 영구 업그레이드 시스템 (강화 탭)

영구 업그레이드는 로비의 **'강화'** 탭에서 진행하며, '캐릭터'와 '등급' 두 가지 하위 탭으로 나뉜다.

---

### 17.1 캐릭터 강화 (캐릭터 탭)
각 캐릭터의 기초 능력치를 영구적으로 향상시킨다.

- **업그레이드 항목**: 데미지, 공격속도, 행운(크리티컬 확률), 크리티컬(치명타 데미지)
- **강화 방식**: **골드**를 소모하여 즉시 적용
- **캐릭터 선택**: 강화 탭 상단의 5개 슬롯에 캐릭터를 장착하여 전투 빌드 구성

예시:
- 방깨비 데미지 강화
- 방깨비 공속 강화
- 도라큘 행운 강화
- 천여귀 크리티컬 강화

---

### 17.2 머지 단계 강화 (등급 탭)
캐릭터의 머지 단계(1~5단계)별 기본 위력을 강화한다.

- **대상**: 1단계부터 5단계까지의 각 머지 레벨
- **효과**: 해당 단계에 도달한 모든 캐릭터의 데미지 증가
- **의도**: 고단계 머지를 성공시켰을 때의 보상을 극대화

---

### 17.4 영구 업그레이드와 카드의 관계
- 영구 업그레이드: 기본 성장
- 카드: 해당 스테이지 한정 빌드
- 동일 영구 업그레이드 상태여도 카드 선택에 따라 결과가 달라짐

---

## 19. 보상 시스템

### 19.1 플레이 중 재화
- 플레이 중 골드 획득
- 획득 골드는 결과 화면에서 정산

---

### 19.2 결과 보상
| 결과 | 보상 |
|---|---:|
| 패배 | 획득한 만큼 100% 지급 |
| 중간 포기 | 획득한 만큼 100% 지급 |
| 스테이지 클리어 | 획득량 x2 지급 |

---

### 19.3 클리어 보너스
스테이지 클리어 시 획득한 재화에 2배 보너스 적용

예시:
- 플레이 중 획득 골드: 1,000
- 클리어 보너스 적용: 2,000 지급

---

## 20. 광고 보상 시스템

### 20.1 결과 화면 광고 보상
결과 화면 하단에 광고 보고 보상 2배 버튼이 존재한다.

---

### 20.2 광고 보상 적용
| 결과 | 기본 보상 | 광고 시청 시 |
|---|---:|---:|
| 패배 / 포기 | 100% | 200% |
| 클리어 | 200% | 400% |

---

### 20.3 의도
- 광고 시청을 통한 추가 보상 제공
- 빠른 성장 유도
- 클리어 시 큰 보상 체감

---

## 21. 광고 노출 구조

### 21.1 비VIP 광고
VIP가 아닌 유저에게는 광고가 노출된다.

- 하단 배너 광고 상시 노출
- 스테이지 클리어 시 광고 노출
- 결과 화면 보상형 광고 선택 가능
- 카드 선택 리롤 시 광고 1회 시청 가능

---

### 21.2 광고 종류
| 광고 종류 | 노출 위치 |
|---|---|
| 배너 광고 | 게임 하단 |
| 전면 광고 | 스테이지 클리어 시 |
| 보상형 광고 | 결과 보상 2배, 카드 리롤 |

---

## 22. VIP 시스템

### 22.1 VIP 혜택
VIP 활성화 시 다음 혜택을 제공한다.

- 하단 배너 광고 제거
- 스테이지 클리어 시 광고 제거
- 광고 없이 상시 2배 보상
- 게임 속도 2배 기능

---

### 22.2 VIP 보상 적용
| 결과 | 일반 기본 | VIP |
|---|---:|---:|
| 패배 / 포기 | 100% | 200% |
| 클리어 | 200% | 400% |

---

### 22.3 VIP 게임 속도
- VIP는 게임 속도 2배 기능 사용 가능
- 반복 플레이 속도 향상
- 파밍 효율 증가

---

## 23. UI 상세 구성

### 23.0 해상도 및 화면 규격

| 항목 | 내용 |
|---|---|
| 기준 해상도 | **900 x 1960 (세로)** |
| 화면 비율 | **약 19.5:9** |
| 맞춤 기준 | **세로(높이) 기준으로 화면 맞춤** |

#### 가로 여백 처리
- 가로가 남는 경우 **좌우 여백 영역 생성 (Pillarbox)**
- 해당 영역은 **배경(BG)만 표시**

#### 규칙
- 실제 게임 플레이 영역은 **900 x 1960 내부로 고정**
- 좌우 여백 영역에는 **게임 요소 배치 금지**

---

### 23.1 강화 탭 (Upgrade)
- **최상단 고정 영역**: 
  - 캐릭터 선택 슬롯 5개 (선택된 캐릭터 이미지, X 버튼으로 제거)
  - 빈 슬롯은 프레임 또는 + 아이콘 표시
  - **규칙**: 반드시 5개 선택 필요 (미달 시 타 탭 이동 시 "캐릭터 5개를 선택해야 합니다" 팝업)
- **하위 탭**: 캐릭터 / 등급 (슬롯 영역은 하위 탭과 무관하게 상단 고정)
- **캐릭터 리스트**: 세로 스크롤, 가로 직사각형 카드 형태
- **강화 팝업**: 캐릭터 이미지/이름, 4종 업그레이드 버튼 및 비용 표시, 즉시 반영

---

### 23.2 카드 탭 (Card)
- **전체 카드 그리드**: 가로 5개 그리드 + 세로 스크롤 방식
- **즐겨찾기**: 자주 사용하는 카드를 최상단에 복제하여 표시 (원본 유지)
- **카드 정보**: 외곽선에 해당 카드의 인게임 등장 확률 표기
- **확률 분배 포인트 표시**: 탭 상단에 `확률 분배 포인트: 남은 포인트 / 총 보유 포인트` 표시
- **카드 상세**: 클릭 시 확대 팝업 표시
  - 현재 등장 확률 (%), 현재 분배 포인트 수치
  - [+] / [-] 버튼으로 포인트 조절 (실시간 확률 재계산 반영)
  - 현재 사용 횟수 / 강화 필요 횟수 표시
  - 강화 가능 시 골드 소모 **성능 강화** 버튼 활성화

---

### 23.3 배틀 탭 (Battle - 기본 화면)
- 스테이지 정보 및 진행 현황
- 보유 재화(골드, 다이아) 표시
- 거대한 **START** 버튼

---

### 23.4 상점 & 미션 탭
- **상점**: 다이아, 패키지, VIP 상품 진열
- **미션**: 일일 미션 및 누적 업적 리스트, 보상 수령 버튼

---

### 23.5 인게임 UI
- **상단**: 몬스터 체력바, 웨이브 타이머, 스테이지 정보
- **중단**: 머지 이펙트 및 봉인 상태 연출
- **하단 입력 영역**: 현재 발사 캐릭터(확대), 다음 큐(물방울 아이콘), 조준선(터치 시 노출), 아이템 버튼
  - **조작**: 터치 유지 시 좌우 이동, 릴리즈 시 위로 발사
- **하단 웨이브 진행 UI**: 현재 스테이지의 웨이브 진행 상황 시각 표시
- **결과 UI**: 획득 골드 정산, 클리어 보너스, 광고 보고 2배 받기 버튼

#### 하단 웨이브 진행 UI 상세
```
□-■-□-■-□-■-□-■-□-■
```
- □ = 일반 웨이브 (카드 선택 없음)
- ■ = 카드 선택이 있는 지점 (1/3/5/7/9웨이브 종료 후)
- 웨이브 클리어 시 해당 위치 위에 체크 표시 이미지를 추가한다

---

### 23.6 카드 선택 팝업 UI

#### 기본 구성
- 카드 2장 표시 (2지선다)
- 리롤 버튼 (선택창당 1회, 다이아 또는 광고)
- 자동 선택 체크박스

#### 자동 선택 체크박스
- 문구: `[ ] 자동 선택(3초)`
- 기본값: 체크 해제

#### 자동 선택 ON 상태
- 카운트다운(3, 2, 1) 팝업 가운데 표시
- 카운트다운 종료 시 랜덤 자동 선택
- 카운트다운 중 유저가 터치하면 해당 카드 즉시 선택 (자동 선택 취소)

## 24. 아트 스타일

### 24.1 캐릭터 스타일
- 귀엽고 단순한 2D 캐릭터
- 두꺼운 검정 외곽선
- 투톤 쉐이딩
- 그라데이션 최소화
- 밝은 색감
- 모바일 캐주얼 게임 느낌

---

### 24.2 캐릭터 디자인 방향
초기 캐릭터 컨셉은 '한국형 고전 귀신/요괴로 오해받는 인간 가족' 느낌으로 설정한다.

- 방깨비 (방망이)
- 구미오 (구씨 미오)
- 유우령 (유씨 우령)
- 미희라 (이름이 미희라)
- 도라큘 (도씨 라큘)
- 강시희 (강씨 시희)
- 천여귀 (천씨 여귀)

단, 현재 시스템상 내부 타입은 고유 캐릭터 타입을 기준으로 관리한다.

---

### 24.3 머지 시각 변화
단계가 높아질수록:

- 크기 증가
- 장식 추가
- 외형 화려함 증가
- 공격 이펙트 강화
- 캐릭터 실루엣 강화

---

## 26. 밸런스 체크 포인트

### 26.1 카드 수치
- S 등급 카드가 너무 자주 뜨면 밸런스 붕괴 가능
- 카드 효과는 강하지만 스테이지 한정이므로 과감한 수치 가능
- 등장 확률이 핵심

---

### 26.2 리롤
- 리롤이 기본 행동이 되면 선택 긴장감 감소
- 카드 선택당 1회 제한 유지
- REROLL 카드와 UI 리롤이 함께 존재하므로 등장 확률 관리 필요

---

### 26.3 STRIKE 카드
- 한 웨이브 1회성이라 누적 붕괴는 없음
- 단, 보스 웨이브 또는 고난도 웨이브에서 영향이 큼
- 등장 확률 또는 적용 타이밍 조절 필요

---

### 26.4 TIME 카드
- 다음 웨이브 1회만 +10초
- 중첩 불가
- 안정 카드로 설계
- 너무 자주 뜨면 긴장감 저하

---

### 26.5 봉인
- 캐릭터를 제거하지 않고 스트레스를 주는 좋은 페널티
- 랜덤성이 강하므로 해제 카드와 함께 밸런스 필요
- 고단계 캐릭터 봉인 시 체감이 큼

---

### 26.6 VIP / 광고
- 일반 유저는 배너 + 클리어 광고 + 보상형 광고
- VIP는 광고 제거 + 상시 2배 + 속도 2배
- 보상 배율이 크므로 경제 밸런스 확인 필요

---
